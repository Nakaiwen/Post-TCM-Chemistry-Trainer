後中醫化學考題練習器 README
================================

工具名稱
--------
後中醫化學考題練習器

英文名稱
------------
Post-TCM Chemistry Trainer

工具簡介
--------
「後中醫化學考題練習器」是一個純前端、單檔 HTML 的化學選擇題練習工具，設計給準備後中醫入學考試、需要反覆練習普通化學與有機化學題型的考生使用。

本工具內建題庫、答題回饋、詳解、錯題複習、收藏題目與學習統計功能。使用者只要用瀏覽器開啟 HTML 檔即可使用，不需要安裝伺服器或資料庫。

主要功能
--------
1. 首頁
   - 顯示學習問候語與練習入口。
   - 提供每日練習、分科練習、錯題複習與收藏題目快速入口。
   - 顯示學習提示，提醒使用者每日練習與針對弱項加強。

2. 每日練習
   - 每輪隨機抽取 10 題。
   - 適合每天快速複習與維持解題手感。

3. 分科練習
   - 可依「普通化學」與「有機化學」分類進行練習。
   - 每個細項主題會顯示題數與該主題目前正確率。
   - 適合針對弱項章節集中訓練。

4. 錯題複習
   - 答錯的題目會自動加入錯題清單。
   - 錯題再次答對後會累積正確紀錄。
   - 連續答對兩次後，題目會自動從錯題清單移除。

5. 收藏題目
   - 可將重要或想再次複習的題目加入收藏。
   - 收藏題目會獨立列入收藏練習區。

6. 即時詳解
   - 作答後立即顯示正確答案與解析。
   - 適合邊練習邊補強觀念。

7. 學習統計
   - 顯示累計作答題數、總正確率、連續學習天數、待複習錯題數。
   - 顯示分科正確率與細項主題正確率。
   - 可清除所有本機練習紀錄。

8. 本機自動儲存
   - 使用瀏覽器 localStorage 儲存學習進度。
   - 目前儲存 key 為：chemPracticeProgress_v1
   - 不需要登入，也不會將資料上傳到伺服器。

目前題庫共約 78 題，分為兩大類：

1. 普通化學：約 42 題

   - 化學計量
   - 氣體定律
   - 熱力學
   - 化學動力學
   - 化學平衡
   - 酸鹼平衡
   - 沉澱平衡
   - 電化學

2. 有機化學：約 36 題

   - 結構

   - 醇／醚／酚

使用方式
--------
1. 將「後中醫化學考題練習器.html」放在電腦或網站資料夾中。
2. 使用 Chrome、Edge、Safari、Firefox 等現代瀏覽器開啟。
3. 進入首頁後，選擇練習模式：
   - 每日練習：隨機 10 題
   - 分科練習：選擇特定章節
   - 錯題複習：重做曾經答錯的題目
   - 收藏題目：複習自己標記的題目
   - 統計：查看學習表現
4. 點選選項作答後，系統會立即標示正確／錯誤並顯示詳解。
5. 點選「下一題」繼續練習。

安裝需求
--------
本工具不需要安裝後端環境。

建議環境：
- 現代桌面或手機瀏覽器
- 支援 JavaScript
- 支援 localStorage

注意事項：
- 若使用無痕模式或清除瀏覽器資料，學習紀錄可能會消失。
- 若換瀏覽器或換裝置，原本的 localStorage 紀錄不會自動同步。
- 工具使用 Google Fonts 載入字體；若離線使用，瀏覽器會改用備援字體。

檔案結構
--------
目前版本為單檔設計：

後中醫化學考題練習器.html
- HTML：畫面結構

- 累計作答題數
- 累計答對題數
- 分科統計
- 主題統計
- 錯題清單
- 收藏清單
- 作答歷史
- 連續學習天數
- 今日作答題數
- 每題答題狀態

儲存 key：
chemPracticeProgress_v1

若要重置進度，可在工具的「統計」頁點選「清除所有紀錄」。

設計風格
--------
整體視覺採用溫潤紙色、朱紅、玉綠與金色作為主色，搭配類書卷與中醫典籍感的排版。畫面重點是安靜、專注、適合長時間練習，不使用過度刺激的遊戲化效果。

適合使用者
----------
- 準備後中醫入學考試的考生
- 想補強普通化學與有機化學基礎的學生
- 需要每日短時間練題的自學者
- 想建立錯題複習循環的考生

維護與擴充建議
--------------
1. 增加題庫
   - 可在 JavaScript 的 QUESTIONS 陣列中新增題目。
   - 每題建議包含 id、cat、topic、q、opts、ans、exp。

2. 題目格式建議
   - id：題目唯一代碼，例如 g043、o037
   - cat：大分類，例如 普通化學、有機化學
   - topic：細項主題，例如 酸鹼平衡、立體化學
   - q：題幹
   - opts：四個選項
   - ans：正確答案索引，0 代表第一個選項
   - exp：詳解文字

3. 上線部署
   - 可直接部署到 GitHub Pages、Netlify、Vercel 或一般靜態網站空間。
   - 因為是純前端工具，不需要資料庫。

4. 未來可新增功能
   - 匯出／匯入學習紀錄
   - 題庫 JSON 外部化
   - 模擬考模式
   - 依錯題頻率排序
   - 依章節設定每日練習比例
   - 手機版安裝為 PWA
   - 題目搜尋功能
   - 難度標籤與考點標籤
